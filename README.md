# Клеточно-автоматная модель оценки выхода Яндекс.Такси на международные рынки

[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## Описание

Данный проект реализует модель клеточного автомата для оценки и прогнозирования выхода компании Яндекс.Такси на международные рынки.

В рамках модели каждая страна рассматривается как отдельная ячейка (клетка), состояние которой определяется набором факторов:

- **Размер рынка** (население)
- **Уровень экономики** (ВВП на душу населения)
- **Распространённость смартфонов**
- **Регуляторные ограничения**
- **Уровень конкуренции**
- **Качество инфраструктуры**
- **Эффект соседства** (наличие компании в соседних странах)

## Функционал

### Реализованный функционал:
- ✅ Расчёт привлекательности рынков по 6 факторам
- ✅ Моделирование эффекта соседства
- ✅ Симуляция выхода на рынки по годам (2015-2024)
- ✅ Сравнение расчётных и фактических сроков выхода
- ✅ Визуализация результатов (3 типа графиков)
- ✅ Расчёт метрики точности модели

## Установка

### Требования:
- Python 3.10 или выше
- pip (менеджер пакетов Python)

### Установка зависимостей:

```bash
# Клонирование репозитория
git clone https://github.com/YOUR_USERNAME/yandex-taxi-ca-model.git
cd yandex-taxi-ca-model

# Установка зависимостей
pip install -r requirements.txt
```

Или установка вручную:
```bash
pip install numpy matplotlib
```

## Запуск

### Основной запуск (с графическим интерфейсом):
```bash
python3 main.py
```

### Запуск без GUI (для серверов):
```bash
MPLBACKEND=Agg python3 main.py
```

### Генерация и сохранение графиков:
```bash
python3 generate_plots.py
```

Графики будут сохранены в папке `screenshots/`.

## Параметры модели

| Параметр | Значение | Описание |
|----------|----------|----------|
| `NEIGHBOR_BONUS` | 0.15 | Максимальный бонус соседства |
